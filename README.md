# training_jclark

A repository for learning git and GitHub in the Arctic Data Center reproducible research course. This course took place over 5 days from February 26 to March 1, 2024, and it was great!

## Project Information

Git is not only a powerful tool for individual work but also an excellent choice for collaborating with friends and colleagues.
Git ensures that after you’ve completed your contributions to a repository, you can confidently synchronize your changes with changes made by others.
